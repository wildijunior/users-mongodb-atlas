# Users List App
Crud using MongoDB Cloud Atlas

* Typescript
* JavaScript
* Bootstrap v5.0.0-beta1
* MongoDB Cloud Atlas
* Mongoose
* EJS
* ExpressJS
* MVC Concept
* Body-Parser
* Dotenv

### Run Project
Choose the language
In the project language folder
```
npm install
```
After install the dependencies: 
```
npm start
```
